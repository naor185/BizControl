# מדריך הגדרות מערכת BizControl — מדריך מפורט

---

## איך מגיעים להגדרות?

סרגל צד ← **הגדרות (⚙️)** → בתוך הדף יש **7 לשוניות** בחלק העליון.
בסוף כל שינוי — לחץ על כפתור **"שמור הגדרות סטודיו"** (שחור, בתחתית המסך).

---

## לשונית 1 — מיתוג ועיצוב 🎨

### לוגו העסק
- **מה זה:** הלוגו שמופיע בממשק ההתחברות ובדפי הנחיתה שהלקוחות שלך רואים.
- **פורמטים מותרים:** PNG (מומלץ עם רקע שקוף), JPEG, SVG.
- **גודל מומלץ:** לפחות 400×400 פיקסל, ברזולוציה גבוהה.
- **איך מעלים:** לחץ "בחר קובץ לוגו" → בחר קובץ → הלוגו נשמר אוטומטית.

## לשונית 2 — דפי נחיתה 🚀

### מחולל עיצוב AI ✨
- **מה זה:** כלי שמקבל תיאור טקסטואלי של העסק שלך ומייצר אוטומטית: צבעים, כותרת ותיאור לדף הנחיתה.
- **מכסה:** **3 שימושים בחודש** (מתאפס ב-1 לכל חודש).
- **איך משתמשים:** כתוב תיאור בשדה → לחץ "צור עיצוב". לדוגמה: *"סטודיו קעקועים בוטיק, אווירה אפלה ואקסקלוסיבית, סגנון מינימליסטי"*.
- **מה מתעדכן אוטומטית:** צבע ראשי, צבע משני, כותרת דף, תיאור דף.

### צבע ראשי (Primary)
- **מה זה:** הצבע הדומיננטי בדף הנחיתה — כפתורים, כותרות, אלמנטים בולטים.
- **פורמט:** קוד HEX (לדוגמה: `#1a1a1a` לשחור כהה, `#c9a96e` לזהב).
- **איך שינוי:** לחץ על ריבוע הצבע לבחירה ויזואלית, או הקלד קוד HEX ישירות.

### צבע משני (Secondary)
- **מה זה:** צבע המשלים — טקסטים, רקעים, אלמנטים משניים.
- **טיפ:** אם הצבע הראשי כהה, הצבע המשני בדרך כלל יהיה בהיר (לבן/שמנת).

### כותרת דף הנחיתה
- **מה זה:** הכותרת הגדולה שהלקוח רואה כשהוא פותח את הלינק להצטרפות למועדון.
- **לדוגמה:** *"ברוכים הבאים לסטודיו XYZ"*, *"הצטרף למועדון הלקוחות שלנו"*.

### תיאור דף הנחיתה
- **מה זה:** פסקת טקסט קצרה מתחת לכותרת — מה מקבל הלקוח בהצטרפות.
- **לדוגמה:** *"נקודות בכל ביקור, הנחות יום הולדת ועוד הטבות בלעדיות"*.

### גופן כותרת / גופן תיאור
- **מה זה:** הפונט שמשמש לכותרת ולטקסט בדף הנחיתה.
- **אפשרויות נפוצות:** Heebo, Assistant, Frank Ruhl Libre (מותאם לעברית).

### תמונת רקע לדף
- **מה זה:** תמונה שמופיעה ברקע של דף הנחיתה.
- **מומלץ:** תמונה רחבה (1920×1080), איכות גבוהה. תמונת עבודה/אווירה של הסטודיו.

### תמונות גלריה (1, 2, 3)
- **מה זה:** עד 3 תמונות שמוצגות בגלריה בדף הנחיתה להציג עבודות/אווירה.
- **מומלץ:** ריבועיות (1:1), לפחות 800×800 פיקסל.

### בחירת תבנית דף נחיתה
- **מה זה:** יש מספר עיצובים מוכנים לדף הנחיתה — ניתן לעבור ביניהם ולראות תצוגה מקדימה בצד ימין.

## לשונית 3 — הודעות אוטומטיות 📩

> כל ההודעות בלשונית זו נשלחות **אוטומטית** על ידי המערכת בנקודות זמן מוגדרות. אין צורך לשלוח ידנית.

### משתני תבנית (placeholders)
בכל הודעה ניתן להשתמש במשתנים שהמערכת מחליפה אוטומטית בפרטים האמיתיים:

| משתנה | מה הוא מכיל |
|---|---|
| `{client_name}` | שם מלא של הלקוח |
| `{appointment_date}` | תאריך התור (לדוגמה: 15/06/2025) |
| `{appointment_time}` | שעת התור (לדוגמה: 15:30) |
| `{appointment_title}` | שם הטיפול/סוג התור |
| `{artist_name}` | שם האמן/ית שמבצע את הטיפול |
| `{deposit_amount}` | סכום המקדמה בשקלים |
| `{bit_link}` | לינק לתשלום בביט |
| `{paybox_link}` | לינק לתשלום בפייבוקס |
| `{bank_details}` | פרטי העברה בנקאית מפורטים |
| `{studio_address}` | כתובת הסטודיו |
| `{map_link}` | לינק לניווט במפה |
| `{portfolio_link}` | לינק לתיק עבודות |
| `{cancellation_free_days}` | מספר ימי ביטול חינמי |
| `{deposit_lock_days}` | מספר ימים לנעילת מקדמה |
| `{contact_phone}` | טלפון ליצירת קשר |
| `{join_link}` | לינק הצטרפות למועדון לקוחות |
| `{points_used}` | נקודות שמומשו |
| `{discount_amount}` | שווי ההנחה בשקלים |
| `{loyalty_points}` | יתרת נקודות הלקוח |
| `{points_on_signup}` | נקודות מתנה בהצטרפות |

### הודעת אישור תור (וואטסאפ / מייל)
- **מתי נשלחת:** מיד כשנוצר תור חדש במערכת.
- **לתורים ללא מקדמה:** נשלחת הודעה פשוטה עם פרטי התור.
- **לתורים עם מקדמה:** נשלחת במקום הודעה זו **הודעת בקשת מקדמה** (ראה למטה).
- **שדה:** "תבנית אישור תור - וואטסאפ" / "תבנית אישור תור - מייל"

### הודעת תזכורת (וואטסאפ / מייל)
- **מתי נשלחת:** יום לפני התור (בלילה, בזמן ריצת המשימות האוטומטיות).
- **שימוש נפוץ:** תזכורת פשוטה עם פרטי התור.
- **שדה:** "תבנית תזכורת - וואטסאפ" / "תבנית תזכורת - מייל"

### הודעת ברוך הבא למועדון (וואטסאפ / מייל)
- **מתי נשלחת:** כשלקוח נרשם לדף הנחיתה ומצטרף למועדון הלקוחות.
- **שימוש נפוץ:** ברכת ברוך הבא + ציון כמות הנקודות שקיבל.
- **שדה:** "תבנית הצטרפות למועדון - וואטסאפ" / "- מייל"

### הודעה אחרי תשלום (וואטסאפ / מייל)
- **מתי נשלחת:** אחרי שמאשרים תשלום ידנית במערכת (לחצן "אישור תשלום" בתור).
- **שימוש נפוץ:** אישור קבלה ותודה.
- **שדה:** "תבנית אחרי תשלום - וואטסאפ" / "- מייל"

### הודעת שינוי תור (וואטסאפ / מייל)
- **מתי נשלחת:** כשמשנים את תאריך/שעה של תור קיים.
- **שדה:** "תבנית שינוי תור - וואטסאפ" / "- מייל"

### הודעת ביטול תור (וואטסאפ / מייל)
- **מתי נשלחת:** כשמבטלים תור במערכת.
- **שדה:** "תבנית ביטול תור - וואטסאפ" / "- מייל"

### הודעת יום הולדת (וואטסאפ / מייל)
- **מתי נשלחת:** ביום ההולדת של הלקוח (המערכת בודקת כל יום).
- **הטבת יום הולדת (%):** אחוז הנחה שניתן ביום ההולדת (לדוגמה: 10 = 10% הנחה).
- **שדה:** "תבנית יום הולדת - וואטסאפ" / "- מייל"

---

### הודעות ייחודיות לוואטסאפ (ללא מייל):

#### הודעת בקשת מקדמה
- **מתי נשלחת:** אוטומטית כשנוצר תור שיש בו **סכום מקדמה > 0**.
- **מטרה:** לבקש מהלקוח לשלם מקדמה לפני אישור התור.
- **כוללת:** פרטי תשלום (ביט, פייבוקס, העברה בנקאית).
- **שדה:** "הודעת בקשת מקדמה - וואטסאפ"

#### הודעת אישור מקדמה (פרטים מלאים)
- **מתי נשלחת:** אוטומטית אחרי שבעל העסק מאשר ידנית שהמקדמה התקבלה (לחצן "אישור מקדמה" בתור).
- **מטרה:** אישור רשמי לתור + כל הפרטים הדרושים.
- **כוללת:** תאריך, שעה, אמן/ית, כתובת, ניווט, תיק עבודות, מדיניות ביטולים.
- **שדה:** "הודעת אישור מקדמה - וואטסאפ"

#### הודעה ללקוח שאינו חבר מועדון
- **מתי נשלחת:** כשמגיע לקוח שאינו רשום במועדון הלקוחות.
- **מטרה:** הזמנה להצטרף עם תמריץ נקודות.
- **שדה:** "הודעה ללקוח שאינו חבר"

#### הודעת מימוש נקודות
- **מתי נשלחת:** כשלקוח מממש נקודות בתשלום.
- **כוללת:** כמה נקודות מומשו, שווי ההנחה, יתרת נקודות.
- **שדה:** "הודעת מימוש נקודות"

## לשונית 4 — מדיניות וכתובת 📋

### כתובת הסטודיו
- **מה זה:** הכתובת הפיזית של הסטודיו.
- **למה חשוב:** נכנסת אוטומטית לתוך `{studio_address}` בהודעת אישור המקדמה.
- **לדוגמה:** *"רחוב הרצל 45, תל אביב"*

### לינק ניווט במפה
- **מה זה:** לינק לגוגל מפות / Waze לסטודיו.
- **איך מקבלים:** גוגל מפות → חפש את הכתובת → כפתור "שתף" → "העתק קישור".
- **נכנס לתוך:** `{map_link}` בהודעות.

### לינק תיק עבודות
- **מה זה:** לינק לאינסטגרם, אתר, או דף שמציג את העבודות שלך.
- **לדוגמה:** `https://www.instagram.com/yourstudio`
- **נכנס לתוך:** `{portfolio_link}` בהודעות.

### שם הבנק / סניף / מספר חשבון
- **מה זה:** הפרטים הבנקאיים של העסק.
- **איך מוצגים בהודעה:** מתחבר אוטומטית לפורמט:
  `שם_בנק | סניף X | חשבון Y`
  ונכנס לתוך `{bank_details}`.

### מדיניות ביטולים — ימי ביטול חינמי
- **מה זה:** כמה ימים לפני התור הלקוח יכול לבטל ולקבל **החזר מלא** של המקדמה.
- **ברירת מחדל:** 7 ימים.
- **לדוגמה:** אם הגדרת 7 — ביטול 8 ימים לפני = החזר מלא. ביטול 3 ימים לפני = ללא החזר.
- **נכנס לתוך:** `{cancellation_free_days}` בהודעות.

### ימי נעילת מקדמה (שינוי תור)
- **מה זה:** כמה ימים לפני התור ניתן **לשנות** (לדחות/להקדים) תור מבלי לאבד את המקדמה.
- **ברירת מחדל:** 7 ימים.
- **לדוגמה:** אם הגדרת 7 — שינוי 8 ימים לפני מועד = בסדר. שינוי 3 ימים לפני = המקדמה ננעלת.
- **נכנס לתוך:** `{deposit_lock_days}` בהודעות.

## לשונית 5 — חוקים ואוטומציה ⚙️

### עיכוב הודעת טיפול אחרי הטיפול (דקות)
- **מה זה:** כמה דקות אחרי סיום תור נשלחת הודעת "אחרי טיפול" (הוראות טיפול, בקשת ביקורת).
- **ברירת מחדל:** 30 דקות.
- **לדוגמה:** אם הגדרת 60 — שעה אחרי תאריך+שעת התור נשלחת ההודעה.

### הודעת טיפול אחרי הסרה (Aftercare)
- **מה זה:** תוכן ההודעה שנשלחת אחרי הטיפול — בדרך כלל הוראות טיפול בקעקוע/פירסינג + בקשה לביקורת.
- **שדה:** "הודעת טיפול אחרי הסרה"

### קישורי ביקורת (Google / Instagram / Facebook / WhatsApp)
- **מה זה:** לינקים לדפי הביקורת שלך ברשתות השונות.
- **כיצד ישמשו:** ניתן לשלב אותם בהודעת ה-Aftercare (עתידי — בשלב זה שמור לפרמטרים).

### נקודות לכל תור שהסתיים
- **מה זה:** כמה נקודות מועדון מקבל הלקוח על כל תור שהסתיים.
- **לדוגמה:** 10 = הלקוח מקבל 10 נקודות על כל ביקור.

### נקודות בהרשמה
- **מה זה:** נקודות מתנה שהלקוח מקבל ברגע שנרשם לדף הנחיתה ומצטרף למועדון.
- **ברירת מחדל:** 50 נקודות.
- **נכנס לתוך:** `{points_on_signup}` בהודעת ברוך הבא.

### אחוז נקודות לפי תשלום
- **מה זה:** אחוז מכל תשלום שמתווסף כנקודות ללקוח.
- **ברירת מחדל:** 5%.
- **לדוגמה:** תשלום של 500₪ עם 5% = 25 נקודות נוספות.

### שעת התחלת יומן / שעת סיום יומן
- **מה זה:** הגדרת שעות הפעילות שמוצגות ביומן התורים.
- **ברירת מחדל:** 08:00 עד 23:00.
- **חשוב:** לא מונע קביעת תורים מחוץ לשעות — רק קובע את תצוגת היומן.

## לשונית 7 — חיבורים (API) 🔌

### ספק שליחת וואטסאפ

#### Green API (מומלץ — עם המספר הקיים שלך)
- **מה זה:** שירות שמאפשר לשלוח וואטסאפ **מהמספר הקיים שלך** (אישי או עסקי) על ידי סריקת QR.
- **עלות:** ~50 ש"ח / חודש.
- **יתרון:** ממשיך לעבוד עם אותו מספר שהלקוחות מכירים. לא צריך מספר נפרד.
- **Instance ID:** המספר המזהה של החשבון שיצרת ב-green-api.com.
- **API Token:** הסיסמה הסודית לגישה ל-Instance.
- **איך מגדירים:**
  1. היכנס לאתר green-api.com
  2. צור חשבון → צור Instance חדש
  3. סרוק QR עם הוואטסאפ שלך
  4. העתק Instance ID + API Token → הכנס בשדות כאן
  5. העתק את **כתובת ה-Webhook** (תיבה כתומה מתחת לשדות) → הכנס ב-Green API → Settings → Webhooks

#### Meta Cloud API (חינמי — מספר נפרד)
- **מה זה:** שירות הוואטסאפ הרשמי של Meta, עם **1,000 הודעות חינם בחודש**.
- **מגבלה:** דורש **מספר טלפון נפרד** שאינו מחובר לאפליקציית וואטסאפ (למשל: סים שני, eSIM).
- **Phone Number ID:** מזהה המספר מפורטל המפתחים של Meta.
- **Access Token:** טוקן גישה קבוע (Permanent Token) מ-Meta for Developers.
- **איך מגדירים:**
  1. כנס ל-developers.facebook.com
  2. צור App חדש → WhatsApp → Business
  3. הוסף מספר טלפון ← קבל Phone Number ID
  4. צור Permanent Access Token
  5. הכנס Phone Number ID + Token כאן
  6. ב-Meta Developers → WhatsApp → Configuration → Webhook:
     - **Callback URL:** העתק מהתיבה הכחולה מטה
     - **Verify Token:** `bizcontrol_verify`

### הגדרות דואר אלקטרוני (SMTP)
- **מה זה:** הגדרות שרת המייל דרכו המערכת שולחת הודעות מייל אוטומטיות.
- **SMTP Host:** כתובת שרת המייל (לדוגמה: `smtp.gmail.com`).
- **SMTP Port:** פורט (בדרך כלל: 587 לTLS, 465 לSSL).
- **SMTP User:** כתובת המייל ממנה נשלחות ההודעות.
- **SMTP Password:** לGmail — יש ליצור "App Password" בהגדרות הגוגל (לא הסיסמה הרגילה!).
- **From Email:** הכתובת שתופיע כשולח.
- **כפתור "שלח מייל בדיקה":** אחרי שמירת הגדרות SMTP — בדוק ששירות המייל עובד.

### Google Calendar
- **מה זה:** חיבור בין יומן BizControl ליומן Google שלך — תורים יסונכרנו אוטומטית.
- **Client ID + Client Secret:** פרטי אפליקציה שיצרת ב-Google Cloud Console.
- **איך מגדירים:**
  1. כנס ל-console.cloud.google.com
  2. צור Project חדש → Enable Google Calendar API
  3. Credentials → Create OAuth Client ID → Application Type: Web Application
  4. Authorized redirect URI: `http://localhost:3000/api/auth/google/callback`
  5. העתק Client ID ו-Client Secret לשדות כאן
  6. שמור → לחץ "חבר יומן Google" → התחבר עם חשבון Google → מאשר הרשאות
- **סטטוס חיבור:** לאחר חיבור מוצלח יופיע "✅ יומן גוגל מחובר".

## שאלות נפוצות

**ש: שיניתי הגדרה — מתי זה ייכנס לתוקף?**
ת: מיד לאחר לחיצה על "שמור הגדרות סטודיו". הודעות הבאות ישתמשו בהגדרות החדשות.

**ש: הלקוח לא קיבל הודעת וואטסאפ — מה לבדוק?**
ת: 1) האם ספק הוואטסאפ מוגדר (לשונית אינטגרציות). 2) האם Instance ID ו-API Token נכונים ונשמרו. 3) בדוק ב-green-api.com שה-Instance פעיל (ירוק). 4) בדוק שה-Webhook מוגדר.

**ש: איך מוסיפים משתנה להודעה?**
ת: פשוט כתוב את שם המשתנה בתוך סוגריים מסולסלים: `{client_name}`. המשתנים הזמינים מוצגים כפתורים בתחתית כל שדה הודעה — לחץ עליהם להכנסה אוטומטית.

**ש: מה ההבדל בין "ימי ביטול חינמי" ל"ימי נעילת מקדמה"?**
ת: **ביטול** = ביטול התור לגמרי ודרישת החזר כסף. **נעילה** = שינוי/דחייה של התור תוך שמירה על המקדמה. שניהם קובעים סף זמן שלפניו הלקוח יכול לפעול ללא עלות.

**ש: איפה רואים את תיבת ההודעות הנכנסות?**
ת: סרגל צד → **תיבת הודעות (💬)**. שם מוצגות כל השיחות הנכנסות מוואטסאפ עם אפשרות לענות ישירות.
